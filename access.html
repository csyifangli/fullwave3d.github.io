Access

To access the confidential portions of Fullwave Game Changer website, first register with GitHub, then contact m.warner@imperial.ac.uk providing a GitHub username to request access.

Register with GitHub here
Select "Sign up for GitHub", chose a username and password, and supply a corporate email address that belongs to one of the FULLWAVE sponsors.

Contact m.warner@imperial.ac.uk, using the same corporate email address, specifying your GitHub username and requesting access to the Fullwave Game Changer site. You will receive email confirmation when access is granted.
