Access the Game Changer website

To access the Game Changer website, you first need to register with github.com, then contact us by email with your GitHub username to request access.

Register with GitHub here, select "Sign up for GitHub", chose a username and password, and supply a corporate email address that belongs to one of the Game Changer sponsors.

Contact us at m.warner@imperial.ac.uk, using the same corporate email address, specifying your GitHub username and requesting access to the Game Changer site. We will add you to the list of permitted users, and email confirmation back to you.
